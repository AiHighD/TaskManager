Task Manager 

Bảng Tasks - Task_Id(key) - task_name - topic - description(mô tả) - StartTime(ngay bat dau) - EndTime(ngay ket thuc) - Status(trang thai hoan thanh)

Bảng Progress(Tiến độ công việc) - Progress_Id(key) - tỉ lệ hoàn thành - note - khóa phụ task id

Bảng Documents(tài liệu) - Documents_Id(key) - Doc - Time_Create(ngày tạo) - khóa phụ task id

Bảng Attachments(Tệp đính kèm) - Attachments_Id(key) - Description( tả tài liệu) - File - Image - ngày tạo - khóa phụ task id



dotnet aspnet-codegenerator controller -name TasksController -m Tasks -dc TasksManager.Data.TasksDbContext --relativeFolderPath Controllers --useDefaultLayout --referenceScriptLibraries

dotnet aspnet-codegenerator controller -name ProgressController -m Progress -dc TasksManager.Data.TasksDbContext --relativeFolderPath Controllers --useDefaultLayout --referenceScriptLibraries

dotnet aspnet-codegenerator controller -name DocumentsController -m Documents -dc TasksManager.Data.TasksDbContext --relativeFolderPath Controllers --useDefaultLayout --referenceScriptLibraries

dotnet aspnet-codegenerator controller -name AttachmentsController -m Attachments -dc TasksManager.Data.TasksDbContext --relativeFolderPath Controllers --useDefaultLayout --referenceScriptLibraries
